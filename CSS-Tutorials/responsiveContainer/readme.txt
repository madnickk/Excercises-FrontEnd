Responsive Boxen

	width: min(800px, 90vw); 

ist eine kurze Schreibweise von 
	
	width: 800px;
	max-width: 90vw;

Es passiert folgendes: 

Die Box wird immer die kleinere Variante anzeigen. Wenn 90vw des Screens kleiner ist als 800 Pixel (bei kleinen Telefonen zum Beispiel) wird 90vw angezeigt. 

Bei großen Display könnte 800px kleiner sein. Dann wird diese Größe verwendet.
